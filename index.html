CHAPTER 01 · FUNDAMENTALS

What is a Python List? 📦

A list is Python's most versatile data structure — an ordered, mutable collection that can hold any mix of data types. Think of it as a magical box that can store anything!

📐 Key Properties

Ordered ✓Mutable ✓ Allows Duplicates ✓Heterogeneous ✓ Zero-Indexed ✓Dynamic Size ✓
# Creating different kinds of lists fruits = ["apple", "banana", "cherry"] numbers = [10, 20, 30, 40, 50] mixed = [1, "hello", 3.14, True, None] empty = [] nested = [[1,2], [3,4]] # 2D list / matrix print(len(fruits)) # 3 print(fruits + ["date"]) # concatenation print([0] * 5) # [0, 0, 0, 0, 0]

CHAPTER 02 · ACCESS

List Indexing 🔢

Every item has a position — an index. Python uses 0-based indexing. Negative indices count from the end!

➡️ Positive Indexing

fruits = ["apple","banana","cherry","date","elderberry"] print(fruits[0]) # "apple" print(fruits[2]) # "cherry" print(fruits[4]) # "elderberry"

⬅️ Negative Indexing

# Count from the END print(fruits[-1]) # "elderberry" print(fruits[-2]) # "date" print(fruits[-5]) # "apple"

🎬 Index Explorer — Click any box or type an index!

fruits[0] = "apple"
⚠️ IndexError: Accessing fruits[10] on a 5-item list throws an IndexError! Valid range: -5 to 4.

✏️ Modifying via Index

fruits[1] = "blueberry" # replace banana print(fruits) # ["apple", "blueberry", "cherry", ...]
💡 Pro tip: Use list[-1] to always grab the last element without knowing the length!
CHAPTER 03 · RANGES

List Slicing ✂️

Slicing extracts a sub-list using list[start : stop : step]. Returns a new list — the original is untouched!

📐 Syntax

lst[start : stop : step] # start: begin index (inclusive, default 0) # stop : end index (exclusive, default end) # step : jump size (default 1)
